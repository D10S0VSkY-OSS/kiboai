# Roadmap

This roadmap outlines the future development plans and upcoming features for KiboAI.

## Core Features
- **A2S (Agents as a Service)**
  - Agent Server implementation
  - Agent Client SDK
- **MCP (Model Context Protocol)**
  - MCP Server implementation
  - MCP Client support

## Observability & Monitoring
- **LangFuse Integration**
  - Full observability for traces and metrics

## User Interface
- **WebChat Integrations**
  - Chainlit support
  - LibreChat support

## Security & Reliability
- **Guardrails AI**
  - Implementation of guardrails for input/output validation

## Data Processing & ETL
- **Firecrawl**
  - Web crawling capabilities
- **Unstructured**
  - Document processing and parsing
- **n8n**
  - Workflow automation integration

## Evaluation & Testing
- **DeepEval / Ragas**
  - RAG evaluation pipelines
- **Promptfoo**
  - LLM testing and evaluation
